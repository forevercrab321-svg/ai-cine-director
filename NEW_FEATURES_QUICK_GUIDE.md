# 🎯 新功能快速使用指南

## 📸 批量生成镜头图片 - 新增选项

### 在哪里找到？
项目详情页面 → 向下滚动 → 找到"批量生成镜头图片"面板

### 新增的选项
- ✅ **5个镜头** - 适合小规模精准控制
- ✅ **30个镜头** - 适合大批量快速生成

### 原有选项（保持不变）
- 3个镜头
- 6个镜头
- 9个镜头
- 12个镜头

### 使用方法
1. 选择要生成的数量（3/5/6/9/12/30）
2. 选择模型（Flux 1.1 Pro 或 Flux Schnell）
3. 点击"🚀 生成前 N 张图"
4. 等待生成完成

### 💡 智能提示
- 如果总镜头数少于选择的数量，系统会自动调整
- 例如：只有8个镜头，选择30时会自动变为8

## 🔒 一键跑通全片物理锁链 - 防重复机制

### 在哪里找到？
项目详情页面 → 顶部按钮栏 → 找到"🚀 一键跑通全片物理锁链"按钮

### 新功能说明
**单次锁定机制** - 点击一次后永久锁定，防止重复扣费

### 按钮状态变化

#### 1️⃣ 初始状态（可点击）
```
🚀 一键跑通全片物理锁链
颜色：紫色渐变
```

#### 2️⃣ 执行中状态（不可点击）
```
⏳ 锁链执行中...
颜色：深蓝色
提示：系统正在生成视频
```

#### 3️⃣ 锁定完成状态（永久不可点击）
```
🔒 全片已锁定完成
颜色：绿色
提示：全片已锁定，不可重复运行
```

### 重要说明
- ✅ **防重复扣费**：即使生成失败，也会保持锁定状态
- ✅ **清晰提示**：尝试再次点击会弹出警告提示
- ✅ **重新开始**：如需重新生成，刷新页面即可

### 如果需要重新生成
```
按 F5 或点击浏览器刷新按钮
重新加载页面后即可再次执行
```

## ⚠️ 常见问题

### Q1: 选择30个镜头，但只生成了20个？
**A**: 这是正常的，说明你的项目只有20个镜头。系统会自动调整为实际数量。

### Q2: 点击"一键跑通"后发现有错误，想重新点击？
**A**: 不能重新点击，系统已锁定防止重复扣费。请刷新页面后重新开始。

### Q3: 为什么生成失败了还要锁定？
**A**: 这是为了保护你的账户安全，防止连续重试导致重复扣费。

### Q4: 能否在生成过程中取消？
**A**: 系统会完成已启动的生成任务，锁定机制仍然生效。

### Q5: 批量生成和一键跑通有什么区别？
**A**: 
- **批量生成**：只生成镜头的图片
- **一键跑通**：生成全片的视频（从第一场到最后一场）

## 🎉 最佳实践

### 建议工作流程

#### 步骤1：生成剧本和场景
1. 输入故事创意
2. 点击"生成故事分镜"

1. 选择适合的数量（5/30等）
2. 点击"🚀 生成前 N 张图"
3. 预览效果

1. 确认所有准备就绪
2. 点击"🚀 一键跑通全片物理锁链"
3. **注意：此操作只能执行一次**
4. 等待全片生成完成
